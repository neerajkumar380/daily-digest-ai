# 📰 Daily Digest AI

An AI-powered news web app that fetches real-time news using a custom backend API.

## 🚀 Features
- 🔍 Search news by keyword
- 🗂 Category-based news (Sports, Tech, Business, etc.)
- 🌙 Dark mode toggle
- 💾 Save news feature
- ⚡ Fast and responsive UI

## 🛠 Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- API: News API
